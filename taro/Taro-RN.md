### 确认系统环境

node 版本 > 8.3

  node --version 

java版本 = 1.8

    javac -version

SDK Manager 中选择  "SDK Platforms" 安装：

    Android SDK Platform 28 （ Android 6.0不一定有Android SDK Platform 28，找其他版本下的Android SDK Platform 28安装即可 ）

    Intel x86 Atom_64 System Image
